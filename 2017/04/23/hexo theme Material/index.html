安装hexo

安装主题

https://material.viosey.com/start/

在站点配置文件里简单布置一下主题，多的先不多折腾了

hexo s 本地就有了

开始创作

hexo n
hexo g
hexo d
如果博客遇到什么不可抗力 hexo c
hexo s 预览

推到网上！

hexo d 报错 ERROR Deployer not found，淡定google
先 npm install hexo-deployer-git –save 再

hexo d 提示输入账号密码，搞定！
