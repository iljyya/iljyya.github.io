Sublime text 3126 中文文件名显示方框

2017-05-16
  • Sublime text

问题根本

Sublime Text 3的显示bug，在高分屏下设定显示放大就会触发。

解决方法

Preferences-Settings-User 加上"dpi_scale": 1.0
但字会变很小，详细解决方法：
https://www.zhihu.com/question/24029280

奇遇

折腾半天发现Sublime text Dev channel 3130解决了这个bug，但是需要巨款注册，不过有特俗渠道：thepiratebay, cracknow.
哈哈，塞翁失马！
