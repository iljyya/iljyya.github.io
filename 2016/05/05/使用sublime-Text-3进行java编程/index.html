编译显示： 编码GBK的不可映射字符【注：需要在前面的基础下】

将上述javaRun.bat修改为：

[plain] view plain copy 在CODE上查看代码片派生到我的代码片
@ECHO OFF  
cd %~dp1  
ECHO Compiling %~nx1.......  
IF EXIST %~n1.class (  
DEL %~n1.class  
)  
javac -encoding UTF-8 %~nx1  
IF EXIST %~n1.class (  
ECHO -----------OUTPUT-----------  
java %~n1  
)
